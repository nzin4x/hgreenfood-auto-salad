# hgreenfood-auto-salad
사내 샐러드 신청 자동화 프로그램
선호 식단을 순서대로 신청하며, 신청 결과를 data.json 에 표출한다.
셀프 스케쥴러이므로, 일단 실행해 두면 알아서 신청한다.

# 사용 방법

## config.user.yaml 파일 수정
- id, pw, 희망 식단 이니셜 목록을 저장
- data.go.kr 에서 API 를 발급 받아 기록
